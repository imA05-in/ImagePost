- **User Authentication**: Secure Google Login integration (Chrome recommended for best experience)
- **Blog Management**: Create, read, and manage blog posts
- **State Management**: Redux Toolkit for efficient state management
- **Form Handling**: React Hook Form for seamless form validation
- **Google OAuth**: Login with google mail (OAuth2 setup)
- **Routing**: Client-side navigation with React Router
- **Backend**: Appwrite for database and authentication services
- **Styling**: Tailwind CSS for responsive, modern UI
- **Build Tool**: Vite for fast development and optimized production builds

## 🛠️ Tech Stack

- **Frontend**: React 19, React Router 7, Redux Toolkit
- **Styling**: Tailwind CSS with Vite integration
- **Backend**: Appwrite
- **Build**: Vite
- **Form Management**: React Hook Form
